# Calculator

A rudimentary online calculator that was made for the Odin Project. Made using HTML, CSS and JavaScript.

## Features
* Addition.
* Subtraction.
* Multiplication.
* Division.
* Exponents.
* Roots.
* Backspace and All Clear button.
* Support of negative numbers.
* Full keyboard support.
